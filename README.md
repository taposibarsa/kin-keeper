👥 Project Name: KeenKeeper | Keep Your Friendships Alive

📌 Short Description

KeenKeeper is a simple friendship management web app where users can keep track of their contacts, monitor interactions (call, text, video), and visualize communication patterns through analytics. It helps users stay connected with friends and maintain relationships effectively.

⚙️ Technologies Used
Next.js (App Router)
React.js
Tailwind CSS
Recharts (for data visualization)
Context API (for state management)

🚀 Key Features
Friend Management – View and manage a list of friends with their contact status, tags, and last interaction time.
Timeline Tracking – Track all interactions (call, text, video) with filtering options.
Analytics Dashboard – Visualize communication data using a pie chart for better insights.
